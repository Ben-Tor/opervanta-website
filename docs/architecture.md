# Opervanta Website Architecture

## Technology Stack

- SvelteKit
- TypeScript
- Tailwind CSS
- pnpm
- Git
- GitHub
- Cloudflare Pages

## Project Principles

- Plan before building
- Keep components reusable
- Avoid unnecessary dependencies
- Commit at meaningful milestones
- Keep structure easy to understand
- Prioritize maintainability over speed

## Initial Folder Direction

The website will use a component-based structure.

Planned folders:

- src/lib/components/ui
- src/lib/components/layout
- src/lib/components/sections
- src/lib/data
- src/lib/utils
- src/routes
